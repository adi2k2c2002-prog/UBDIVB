
Universal Bank - Streamlit dashboard package
-------------------------------------------

Files included:
- app.py : Main Streamlit application. Run with `streamlit run app.py`.
- sample_universal_bank.csv : A synthesized sample dataset compatible with Universal Bank columns.
- README.md : This file.

Instructions to deploy:
1. Create a new GitHub repository and upload all files from this package's root.
2. On Streamlit Cloud, connect the GitHub repo and set the main file to `app.py`.
3. (Optional) If you want to use your real dataset, upload it through the app 'Upload' control or place it in the repo and adjust the code.
4. Use the 'Model Trainer' tab to train models and view metrics; then use the 'Bulk Predict' tab to upload new data and get predictions.
